## Running the App
### Server
The app must be hosted on a machine on the same network as the mobile device accessing the app.

1. Download the [latest release](https://github.com/mbuck29/JayWalk/releases/latest) from this repository

2. Open a terminal in the root folder

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
   npx expo start
   ```

### Client
1. Download the [Expo Go](https://expo.dev/go) app for your mobile device

2. Scan the qr code given by the server using your device's qr reader

3. Connect to the app in the Expo Go app
